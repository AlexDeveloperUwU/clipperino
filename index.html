Editor — How it works

  1. 1Import CSV — load a transcript file using the button on the top right.
  2. 2Select lines — click a line, or hold Shift to select several at once.
  3. 3Create a clip — name your selection and save it. Saved clips appear in the side panel.
  4. 4Export — download your clips as JSON, Markdown or EDL, or use Clear Data to start over.

Preview — How it works

  1. 1Load Video — pick the video that matches your transcript.
  2. 2Scrub the timeline — drag the playhead, or use the zoom controls to look closer.
  3. 3Jump to a clip — click any clip block on the timeline to jump straight to it.
  4. 4Subtitles — they follow playback automatically and can be toggled on or off.

Viewer — How it works

  1. 1Import JSON — load a previously exported clips file to review it.
  2. 2Browse clips — each clip shows its name, timing and transcript lines.
  3. 3Reset — use Clear Data to remove the loaded clips.
